Il software di pianificazione della missione consente ai piloti di programmare voli automatizzati, che consentono voli precisi e ripetibili, ideali per la mappatura. Ti consigliamo di utilizzare un software di pianificazione della missione durante l'acquisizione dei dati da utilizzare con WebODM.

WebODM non dispone (ancora) di un pianificatore di missione ufficiale, ma di seguito puoi trovare un elenco di software di terze parti.

## Elenco dei software

| Nome | Costo | Piattaforma | Collegamento | Note |
| -------------- | ------ | ---------- | --------------------------------------------------------------- | ---------------------------------------------------------------- |
| QGroundControl | Gratuito | Mavlink | [qgroundcontrol.com](https://qgroundcontrol.com) | Software open source |
| YMapper | Gratuito | DJI, Litchi| [ymapper.yarosfpv.com](https://ymapper.yarosfpv.com/) | Software open source |
| MavenPilot | Ibrido | DJI | [mavenpilot.com](https://mavenpilot.com/planner/) | Il pilota è a pagamento, ma il pianificatore è gratuito |
| DroneLink | Pagato | DJI, Autel | [dronelink.com](https://dronelink.com) | Sconto disponibile tramite [Lightning](https://webodm.net/dronelink) |
| Litchi | Pagato | DJI | [flylitchi.com](https://flylitchi.com/) | [Litchi Pilot beta](https://forum.flylitchi.com/t/open-beta-litchi-pilot/10621) è gratuito |
